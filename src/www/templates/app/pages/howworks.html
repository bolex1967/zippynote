Cockktail is an online party constructor, through which you can effortlessly organize and promote an amazing party in a variety of formats

from a stag-party in a small circle of friends to an open air music festival.

For party makers
  • define the budget
  • set the type of the party
  • choose location
  • select vendors
  • chat with vendors
  • send invitations/sell tickets
  • enjoy the party!
For clubbers
  • define the budget
  • set the type of the party
  • choose location
  • select vendors
  • chat with vendors
  • send invitations/sell tickets
  • enjoy the party!
For vendors
  • define the budget
  • set the type of the party
  • choose location
  • select vendors
  • chat with vendors
  • send invitations/sell tickets
  • enjoy the party!
